# 功能矩阵

状态只使用“已完成并测试”“已实现但需外部配置”“尚未实现”。

| 功能 | 状态 | 说明 |
| --- | --- | --- |
| 私有登录、会话、CSRF、限流与一次性 Owner 初始化 | 已完成并测试 | 已接入 D1 仓储、生产/开发 Cookie、受保护 HTTP 路由和登录界面。 |
| 邀请令牌、接受页面与作品权限矩阵 | 已完成并测试 | 令牌只保存摘要；平台邀请限 Owner，作品邀请限作品拥有者；接受操作使用原子预留标记。 |
| D1 schema、作品、卷章目录与工作台 | 已完成并测试 | 包含两份幂等迁移、参数化查询、建书、章节创建/重命名/移动和元数据列表。生产需 `DB` binding。 |
| 三栏写作、响应式抽屉、实时字数和 Tiptap 标准正文 | 已完成并测试 | 已用真实 Edge 浏览器检查桌面、1080px 平板及 390px 手机视口。 |
| 用户隔离的加密离线草稿和幂等同步队列 | 已完成并测试 | 每用户 32 字节 DEK，服务端 KEK 封装；草稿和队列均为 AES-GCM 密文。 |
| 云端保存、自动快照、版本恢复和冲突副本 | 已完成并测试 | 重复操作 ID 不重复保存；恢复/采用冲突副本前保存当前版本。 |
| 私人备注、协作批注和修改建议 | 已完成并测试 | 三类数据分离；Viewer 只读，Commenter 可建议，Owner/Editor 才能接受建议。 |
| 回收站、恢复和永久删除 | 已完成并测试 | 永久删除按外键顺序清理数据并删除对应私有 R2 对象。生产需 `OBJECTS` binding。 |
| R2 私有文件与加密备份服务 | 已实现但需外部配置 | DOCX/备份类型、所有权、大小和对象键已受控；需正式 R2 与 `MOJIE_BACKUP_MASTER_KEY` 做生产探针。 |
| 旧 IndexedDB 迁移服务 | 已完成并测试 | 具备确认、备份优先和 `migration_id` 幂等服务；浏览器迁移向导尚未实现。 |
| WebDAV/S3 外部备份 | 尚未实现 | 等待实际账号后实现上传、读取、删除探针和适配器。 |
| 模板、校对、敏感词、大纲人物、地图图谱、榜单 | 尚未实现 | 后续产品阶段；榜单采集等待授权网址。 |
| TXT/Markdown/DOCX 项目导入导出 | 尚未实现 | R2 仅提供受控对象边界，尚无用户导入导出流程。 |
| 平台发布、实时多人同段编辑、AI 辅助 | 尚未实现 | 不实现未授权自动发布、验证码绕过或付费正文抓取。 |
