## 身份与密码安全

- 独立认证模块：`scripts/mojie-auth-api.mjs`；
- PBKDF2-SHA256；
- 每个密码独立随机盐；
- Cloudflare Workers 支持范围内的 100,000 次迭代；
- 哈希存储包含算法与迭代版本：`pbkdf2-sha256$100000$...`；
- 登录使用恒定密码派生路径，降低账号是否存在造成的响应差异；
- 摘要比较采用恒定时间累积比较；
- 会话 Cookie 使用 HttpOnly、Secure、SameSite=Strict；
- 首次初始化仅在用户表为空且提供正确管理密钥时允许执行。

Worker 入口会优先且完整地处理所有 `/api/auth/*` 请求，因此线上认证只经过独立兼容模块。打包验证要求该模块存在并检查其 KDF 上限。

## DOCX 本地文件库

- 标准 OOXML ZIP 包可以导入；
- 未修改时导出字节与原文件完全一致，SHA-256相同；
- 修改正文后保留样式、页眉、媒体、关系和其他未编辑包部件；
- 混合粗体/斜体运行节点继续保留；
- 段落数量变化会退出原格式保证范围；
- 原件与编辑件可按当前账号及作品保存在浏览器 IndexedDB；
- 支持载入、下载与删除本机记录；
- 清除浏览器站点数据会删除本机 DOCX，因此界面明确提示手动下载或配置其他备份；
- 未绑定 R2 时云上传返回明确 503，不伪装成远程存储成功。

## 服务端、隐私与自动任务

- 私人内容先经过 `mojie-privacy-guard.mjs`；
- 云端作品列表只返回有效 `work_members` 成员关系；
- 全局 Owner/Admin 不会自动获得其他作者正文权限；
- 新作品只允许具备创作角色的已登录用户创建；
- PWA Service Worker 排除 `/api/`、导航请求和私人 HTML；
- Cron 保留排行榜采集和第三方备份到期处理；
- WebDAV/S3 凭据使用 `MOJIE_BACKUP_MASTER_KEY` 加密后写入 D1；
- 没有第三方存储时不会建立伪造备份记录。
